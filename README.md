Projeto InfoSoc - Informática para Análise de Dados Sociais

Esse repositório contém a homepage do projeto InfoSoc, com informações sobre o próprio projeto e as atividades realizadas.
O projeto InfoSoc é uma iniciativa do Prof. Ronaldo Baltar, docente da Universidade Estadual de Londrina, para disseminar e capacitar estudantes e pesquisadores sobre o uso de ferramentas de informática para a análise de dados sociais.

Coordenador: Prof. Ronaldo Baltar
